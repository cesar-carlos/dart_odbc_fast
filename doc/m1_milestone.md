Criar binding funcional entre Rust e Dart via FFI, permitindo conexão básica a bancos ODBC.

## Entregas

- ✅ Rust ODBC core implementado

- ✅ Bindings Dart gerados automaticamente
- ✅ API Dart de alto nível (Clean Architecture)
- ✅ Testes de integração funcionais

## Funcionalidades

- Inicialização de ambiente ODBC
- Conexão a banco de dados
- Desconexão
- Tratamento de erros básico
